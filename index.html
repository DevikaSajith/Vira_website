VIRA

Gender-Sensitive Emergency Communication Wearable

Project Overview

VIRA is a low-cost, offline, one-touch wearable emergency alert system designed specifically for women in disaster relief or crisis environments. It enables women to communicate their needs discreetly, reliably, and quickly without the need for mobile networks, internet, or verbal communication.

The wearable device sends alerts via LoRa communication to a nearby base unit. The base unit receives, logs, and forwards this information to a centralized dashboard or application for responders to take immediate action. The device provides real-time acknowledgment (ACK) feedback to the user to confirm that their request has been received.

Key Features

Offline First

Works without mobile networks or internet, ideal for crisis zones, ensuring reliability in remote or disaster-stricken areas.

One-Touch Alerts

Simple, discreet button presses for different emergency types (e.g., medical, food, safety), simplifying rapid response.

LoRa Communication

Utilizes long-range, ultra-low power wireless transmission for reliable alerts, extending communication reach significantly.

Real-time ACK

Immediate vibration and LED feedback confirm alert reception to the user, building trust and reducing anxiety.

Gender-Sensitive Design

Empowers women with discreet and reliable communication, addressing specific safety needs in vulnerable environments.

Scalable System

Designed to scale from a few devices to thousands with distributed gateways, supporting large-scale humanitarian efforts.

VIRA System Architecture

1. Wearable Device

(ESP32 + LoRa Module)

  • Inputs: SOS Button
  • Outputs: LED Indicators (Alert Sent / Network Status)
  • Communication: LoRa (Mesh / Point-to-Point)

2. LoRa Mesh Network

(ESP32/Heltec LoRa Devices)

  • Relay SOS messages
  • Optional LoRa Gateway (for cloud relay if internet available)

3. Backend (Optional)

(Firebase Realtime Database / Firestore)

  • SOS Alerts Storage
  • Volunteer/Stock Management
  • Firebase Authentication (User/Admin Login)

4. Mobile/Web App

(Flutter)

  • Emergency SOS Alerts
  • Volunteer Dashboard
  • Contacts Directory
  • Stock Management
  • Real-time Notifications
  • Admin Controls
